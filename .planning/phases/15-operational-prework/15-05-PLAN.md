Output: 'Triarch Dev' OAuth client now has 3+ authorized redirect URIs: admin prod, portal prod, and localhost:3002 dev.

Likely HUMAN-NEEDED step: gcloud's `iam oauth-clients` subcommand has limited update support for redirect URIs. The reliable path is GCP Console manual edit. This plan plans for the Console path as the primary, and tries gcloud as a courtesy fallback.
