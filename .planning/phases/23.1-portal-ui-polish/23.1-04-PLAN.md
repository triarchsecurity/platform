<objective>
Ship the staff "preview as customer" toggle (UX-04) — staff can flip a 1-hour cookie that visually flips the portal UI to customer mode for end-to-end testing without needing a separate customer account.

Purpose: Mike (staff) needs to verify customer-facing UX changes (the changes from plans 01-03 of this phase, plus all future portal work) without signing into a customer account. Currently the portal renders staff banners and suppresses customer action buttons for any staff user, so staff can't visually confirm "this is what a customer sees." This plan adds a reversible, time-limited UI suppression toggle.

**Critical security boundary:** The preview cookie ONLY SUPPRESSES UI for users who already have staff role. It does NOT grant any permission. A non-staff user setting the cookie sees zero change. Server-side mutations remain gated by their existing auth (membership / role checks); the cookie is purely a UI bypass-the-staff-bypass affordance.

Output:
- Server-side cookie module with 1-hour TTL, `__Host-` prefix in production, HttpOnly + SameSite=Lax + Secure-in-prod.
- API route POST (set cookie, staff-only) + DELETE (clear cookie, anyone — clearing is harmless).
- PreviewModeBanner component (client) — fixed top, shows minutes remaining, has Exit preview link.
- StaffCallout modified — when cookie set AND user is staff, returns null (PreviewModeBanner takes over). Otherwise unchanged but gains a "Preview as customer →" link.
- All page.tsx files that derive userRole from `ctx.isStaff` are updated to a helper that respects the preview cookie: `await isPreviewingAsCustomer() ? membership-only : staff-or-membership`.
- Vitest: cookie helper unit tests; route handler tests (POST staff/non-staff/no-session; DELETE always 200); StaffCallout staff-with-cookie / staff-without-cookie / non-staff cases; PreviewModeBanner mounted-with-cookie / mounted-without-cookie cases; **explicit security test: non-staff user with cookie set sees identical UI to non-staff user without cookie set**.
- Portal MINOR bump (new feature) v0.4.x → v0.5.0.
</objective>

       ```typescript
       import { cookies } from 'next/headers';
       import { getPortalSession } from '@/lib/session';

       export const PREVIEW_COOKIE_TTL_MS = 60 * 60 * 1000;
       export const PREVIEW_COOKIE_NAME =
         process.env.NODE_ENV === 'production'
           ? '__Host-portal-preview-as-customer'
           : 'portal-preview-as-customer';

       export function buildSetCookieAttributes() {
         const expiry = Date.now() + PREVIEW_COOKIE_TTL_MS;
         return {
           name: PREVIEW_COOKIE_NAME,
           value: String(expiry),
           httpOnly: true,
           sameSite: 'lax' as const,
           path: '/',
           maxAge: Math.floor(PREVIEW_COOKIE_TTL_MS / 1000),
           ...(process.env.NODE_ENV === 'production' ? { secure: true } : {}),
           // NO domain key — __Host- requires absence
         };
       }

       export function buildClearCookieAttributes() {
         return {
           name: PREVIEW_COOKIE_NAME,
           value: '',
           path: '/',
           maxAge: 0,
           ...(process.env.NODE_ENV === 'production' ? { secure: true } : {}),
         };
       }

       /**
        * Returns true ONLY when cookie is present AND user is staff.
        * Security boundary: non-staff users with the cookie set get false.
        */
       export async function isPreviewingAsCustomer(): Promise<boolean> {
         const session = await getPortalSession();
         if (!session?.user?.isStaff) return false;
         const c = await cookies();
         return !!c.get(PREVIEW_COOKIE_NAME)?.value;
       }

       /**
        * Returns expiry epoch ms when previewing, else null.
        */
       export async function getPreviewExpiry(): Promise<number | null> {
         const session = await getPortalSession();
         if (!session?.user?.isStaff) return null;
         const c = await cookies();
         const v = c.get(PREVIEW_COOKIE_NAME)?.value;
         if (!v) return null;
         const n = Number(v);
         if (!Number.isFinite(n)) return null;
         return n;
       }
       ```
