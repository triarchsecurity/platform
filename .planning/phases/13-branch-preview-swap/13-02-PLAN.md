<objective>
Build the two API routes that drive Phase 13's branch preview swap: a POST that acquires a DB lock and dispatches a FAH rollout, and a GET that polls FAH state, surfaces it, and auto-clears the lock on terminal state (or 8-min timeout).

The atomic UPDATE-with-WHERE-IS-NULL race guard is **the** concurrency mechanism — directly mirroring the PROM-04 pattern from plan 09-03 (`isNull(releaseLogs.promotionDispatchedAt)` → here `isNull(projects.previewBranchLocked)`). Two simultaneous callers cannot both acquire the lock; exactly one's UPDATE returns a row, the other's returns [] and gets a 409. This is the same kind of guard 09-03 used to prevent double-promote between web and Slack — applied here to prevent double-swap between two customer admins clicking simultaneously, or one admin clicking and another in-flight CI push (Pitfall 1 from PITFALLS.md).

The 8-minute timeout in the GET route closes Pitfall 2 (FAH rollout stuck in PENDING/CREATING): if `now - previewBranchLockedAt > 480_000ms`, force-clear the lock and return `state='timeout'`. Without this, a permanently-stuck FAH rollout would lock the project's preview slot indefinitely.

The terminal-state auto-clear (PREV-06) uses a branch-guarded UPDATE — `WHERE key=$slug AND previewBranchLocked=$branch_param` — so that if a stale GET poll sees the OLD swap as terminal at the same moment a new swap acquires a fresh lock, the stale poll's clear cannot wipe the new lock. This is the same defensive pattern the GREEN/RED state machines have used since v2.0.

Auth model: customer admin on this project OR staff (per CONTEXT.md). Existing `getCurrentUserContext` returns `{ email, isStaff, memberships: [{ project_key, role }] }` — same primitive used by the customer releases page itself. We do NOT introduce a new auth helper; we reuse the existing primitives in line.

Backend ID convention: `<slug>-dev` (e.g. `tmi` → `tmi-dev`). For Phase 13 we use this convention directly per CONTEXT.md ("Defer — try the convention first"). If a project's actual backend ID diverges, that surfaces as a 502 fah_dispatch_failed at runtime; user can correct via dashboard or we add an explicit `projects.dev_fah_backend_id` column in a future phase. Location is `us-central1` (the only FAH region used by triarch projects today).

Project ID resolution: read `projects.firebaseProjectId` (existing column from v2.0). For customer-owned projects this is `triarch-dev-{customer}` (the dev clone). The release-promoter SA has IAM grants on those projects (operational setup in 13-01's user_setup).

The `metadata` jsonb column on `projects` is used to store ephemeral per-swap state: `previewRolloutName` (the FAH resource path) and `previewLockedBy` (the actor email). These are written using the `jsonb_set(coalesce(metadata, '{}'), '{key}', $val::jsonb, true)` SQL template established in `release-promotion.ts` — never overwriting the entire metadata column (Pitfall 11 / standing constraint from STATE.md: "no structural mutations to existing columns").

Purpose: PREV-03 (FAH rollout dispatch) + PREV-05 (8-min timeout in polling route) + PREV-06 (terminal-state auto-clear). Plan 13-03 builds the UI on top of the response contracts defined here.
Output: Two production-ready route handlers with full Vitest coverage (15+ test cases total).

async function authForProject(slug: string): Promise<
  | { ok: true; email: string; isStaff: boolean }
  | { ok: false; status: 401 | 403 | 404; body: { error: string } }
> {
  const session = await getServerSession(authOptions);
  if (!session?.user?.email) return { ok: false, status: 401, body: { error: 'unauthorized' } };
  const ctx = await getCurrentUserContext(session);
  if (!ctx) return { ok: false, status: 401, body: { error: 'unauthorized' } };

  const [project] = await db.select({ key: projects.key }).from(projects).where(eq(projects.key, slug));
  if (!project) return { ok: false, status: 404, body: { error: 'not_found' } };

  const membership = ctx.memberships.find(m => m.project_key === slug);
  const isAdmin = ctx.isStaff || membership?.role === 'admin';
  if (!isAdmin) {
    // Per GATE-01 precedent (no project-existence leak): non-members get 404, members-but-not-admin get 403
    if (!membership && !ctx.isStaff) return { ok: false, status: 404, body: { error: 'not_found' } };
    return { ok: false, status: 403, body: { error: 'forbidden' } };
  }

  return { ok: true, email: ctx.email, isStaff: ctx.isStaff };
}

    import { NextRequest, NextResponse } from 'next/server';
    import { getServerSession } from 'next-auth';
    import { authOptions } from '@/lib/auth';
    import { getCurrentUserContext } from '@/lib/auth-context';
    import { db } from '@/lib/db';
    import { projects } from '@/db/schema';
    import { eq, and, isNull, sql } from 'drizzle-orm';
    import { createFahRollout } from '@/lib/fah-rollout';

    const BRANCH_REGEX = /^[a-zA-Z0-9/_.\-]{1,256}$/;
    const FAH_LOCATION = 'us-central1';

    async function authForProject(slug: string) { /* see <auth_helper> in plan context */ }

    export async function POST(req: NextRequest, { params }: { params: Promise<{ slug: string }> }) {
      const { slug } = await params;
      const auth = await authForProject(slug);
      if (!auth.ok) return NextResponse.json(auth.body, { status: auth.status });

      let body: { branch?: unknown };
      try { body = await req.json(); } catch { body = {}; }
      const branch = typeof body.branch === 'string' ? body.branch : '';
      if (!BRANCH_REGEX.test(branch)) {
        return NextResponse.json({ error: 'invalid_branch' }, { status: 400 });
      }

      // Atomic lock acquisition — Pitfall 1 / mirror PROM-04 pattern
      const now = new Date();
      const locked = await db
        .update(projects)
        .set({ previewBranchLocked: branch, previewBranchLockedAt: now })
        .where(and(eq(projects.key, slug), isNull(projects.previewBranchLocked)))
        .returning({ key: projects.key, firebaseProjectId: projects.firebaseProjectId });

      if (locked.length === 0) {
        // Race lost — re-read to surface lock holder
        const [current] = await db
          .select({
            previewBranchLocked: projects.previewBranchLocked,
            previewBranchLockedAt: projects.previewBranchLockedAt,
            metadata: projects.metadata,
          })
          .from(projects)
          .where(eq(projects.key, slug));
        const md = (current?.metadata ?? {}) as Record<string, unknown>;
        return NextResponse.json({
          error: 'lock_held',
          current_branch: current?.previewBranchLocked ?? null,
          locked_at: current?.previewBranchLockedAt?.toISOString() ?? null,
          locked_by: typeof md.previewLockedBy === 'string' ? md.previewLockedBy : null,
        }, { status: 409 });
      }

      const [project] = locked;
      const projectId = project.firebaseProjectId;
      if (!projectId) {
        // Defensive: no Firebase project id on row → release lock + 500
        await db.update(projects)
          .set({ previewBranchLocked: null, previewBranchLockedAt: null })
          .where(eq(projects.key, slug));
        return NextResponse.json({ error: 'project_misconfigured', detail: 'firebaseProjectId is null' }, { status: 500 });
      }
      const backendId = `${slug}-dev`;

      const result = await createFahRollout({ projectId, location: FAH_LOCATION, backendId, branch });
      if (!result.ok) {
        // Release the lock — must NOT leave it held when dispatch failed
        await db.update(projects)
          .set({ previewBranchLocked: null, previewBranchLockedAt: null })
          .where(and(eq(projects.key, slug), eq(projects.previewBranchLocked, branch)));  // branch-guarded so we don't clear a different lock
        return NextResponse.json({ error: 'fah_dispatch_failed', detail: result.error }, { status: 502 });
      }

      // Stamp metadata with rolloutName + lockedBy via jsonb_set (don't overwrite metadata)
      await db.update(projects)
        .set({
          metadata: sql`jsonb_set(jsonb_set(coalesce(${projects.metadata}, '{}'::jsonb), '{previewRolloutName}', ${JSON.stringify(result.rolloutName)}::jsonb, true), '{previewLockedBy}', ${JSON.stringify(auth.email)}::jsonb, true)`,
        })
        .where(eq(projects.key, slug));

      return NextResponse.json({
        rolloutName: result.rolloutName,
        state: result.state,
        locked_at: now.toISOString(),
        locked_by: auth.email,
      }, { status: 202 });
    }

    // src/app/api/projects/[slug]/branch/preview/status/route.ts
    import { NextRequest, NextResponse } from 'next/server';
    import { getServerSession } from 'next-auth';
    import { authOptions } from '@/lib/auth';
    import { getCurrentUserContext } from '@/lib/auth-context';
    import { db } from '@/lib/db';
    import { projects } from '@/db/schema';
    import { eq, and } from 'drizzle-orm';
    import { getFahRolloutState } from '@/lib/fah-rollout';

    const TIMEOUT_MS = 8 * 60 * 1000;  // 8 minutes — Pitfall 2 hard cap

    async function authForProject(slug: string) { /* same helper as POST route */ }

    export async function GET(_req: NextRequest, { params }: { params: Promise<{ slug: string }> }) {
      const { slug } = await params;
      const auth = await authForProject(slug);
      if (!auth.ok) return NextResponse.json(auth.body, { status: auth.status });

      const [project] = await db
        .select({
          previewBranchLocked: projects.previewBranchLocked,
          previewBranchLockedAt: projects.previewBranchLockedAt,
          metadata: projects.metadata,
        })
        .from(projects)
        .where(eq(projects.key, slug));

      if (!project) return NextResponse.json({ error: 'not_found' }, { status: 404 });

      // No lock → idle
      if (!project.previewBranchLocked || !project.previewBranchLockedAt) {
        return NextResponse.json({
          branch: null, state: 'idle',
          locked_at: null, locked_by: null, started_at: null, terminal: true,
        });
      }

      const branch = project.previewBranchLocked;
      const lockedAt = project.previewBranchLockedAt;
      const md = (project.metadata ?? {}) as Record<string, unknown>;
      const rolloutResourcePath = typeof md.previewRolloutName === 'string' ? md.previewRolloutName : undefined;
      const lockedBy = typeof md.previewLockedBy === 'string' ? md.previewLockedBy : null;
      const ageMs = Date.now() - lockedAt.getTime();

      // 8-min timeout — Pitfall 2 hard cap, must run BEFORE FAH poll
      if (ageMs > TIMEOUT_MS) {
        await db.update(projects)
          .set({ previewBranchLocked: null, previewBranchLockedAt: null })
          .where(and(eq(projects.key, slug), eq(projects.previewBranchLocked, branch)));
        return NextResponse.json({
          branch, state: 'timeout', locked_at: lockedAt.toISOString(), locked_by: lockedBy,
          started_at: lockedAt.toISOString(), terminal: true,
        });
      }

      // No rollout name in metadata → can't poll; treat as still in-flight (early state right after POST)
      if (!rolloutResourcePath) {
        return NextResponse.json({
          branch, state: 'PENDING', locked_at: lockedAt.toISOString(), locked_by: lockedBy,
          started_at: lockedAt.toISOString(), terminal: false,
        });
      }

      const result = await getFahRolloutState(rolloutResourcePath);
      if (!result.ok) {
        // FAH poll error: surface, do NOT clear lock — the next poll might succeed
        return NextResponse.json({
          branch, state: 'PENDING', locked_at: lockedAt.toISOString(), locked_by: lockedBy,
          started_at: lockedAt.toISOString(), terminal: false,
          errorMessage: `FAH poll error: ${result.error}`,
          rolloutResourcePath,
        });
      }

      const terminal = ['SUCCEEDED', 'FAILED', 'CANCELLED'].includes(result.state);
      if (terminal) {
        // Branch-guarded clear — stale-poll guard so we don't wipe a NEWER lock
        await db.update(projects)
          .set({ previewBranchLocked: null, previewBranchLockedAt: null })
          .where(and(eq(projects.key, slug), eq(projects.previewBranchLocked, branch)));
      }

      return NextResponse.json({
        branch, state: result.state,
        locked_at: lockedAt.toISOString(), locked_by: lockedBy,
        started_at: lockedAt.toISOString(),
        terminal,
        ...(result.errorMessage ? { errorMessage: result.errorMessage } : {}),
        rolloutResourcePath,
      });
    }
