<objective>
Extend the existing `POST /api/platform/promote-callback` endpoint to post a Slack threaded reply when promote-branch.yml reports its result (RC-06 for conflict; D-13 adds symmetric replies for merged + ci_failed). The reply targets the original `:rocket: Workflow dispatched: promote-branch.yml ...` message that Plan 06-01 anchored via `releaseLogs.metadata.dispatch.{slackChannelId, slackMessageTs}`.

Per D-11, missing `metadata.dispatch` MUST NOT fail the 201 — `promote_attempts` row is the source of truth; the Slack reply is best-effort.

Purpose: Without RC-06, customers who hit a rebase conflict get no feedback in Slack. The :warning: reply is the loop-closer that lets staff act ("Rebase manually on main, push as a new RC to retry.").

- `src/app/api/platform/promote-callback/route.ts`: imports added (`releaseLogs`, `desc`, `and`, `eq`, `postSlackThreadedReply`), release lookup added, `buildPromoteReplyText` helper added, threaded reply call wrapped in try/catch

    ```typescript
    /**
     * Build the threaded Slack reply text for a promote-branch.yml result.
     * Per D-12 (conflict) and D-13 (merged / ci_failed).
     */
    function buildPromoteReplyText(
      result: PromoteResult,
      branch: string,
      mergeSha: string | null,
      conflictFiles: string[],
      ciRunUrl: string | null
    ): string {
      if (result === 'merged') {
        const sha = mergeSha ? mergeSha.slice(0, 7) : 'unknown';
        return `:white_check_mark: Promoted ${branch} to main (sha: ${sha})`;
      }
      if (result === 'ci_failed') {
        return `:no_entry: CI failed for ${branch} — see ${ciRunUrl ?? 'CI logs'}`;
      }
      // result === 'conflict'
      const cap = 50;
      const shown = conflictFiles.slice(0, cap);
      const overflow = conflictFiles.length - cap;
      const fileList = shown.join('\n');
      const overflowLine = overflow > 0 ? `\n+ ${overflow} more files` : '';
      return `:warning: Cannot promote ${branch} — conflicts with main:\n\`\`\`\n${fileList}${overflowLine}\n\`\`\`\nRebase manually on main, push as a new RC to retry.`;
    }
    ```
