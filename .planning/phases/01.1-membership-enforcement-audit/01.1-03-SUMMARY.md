# Phase 01.1 Plan 03: Migrate 23 Staff-Only Endpoints to requireStaff Summary

**One-liner:** Swapped `requireAdmin()` for `requireStaff()` across all 23 staff-only platform endpoints, closing the 403-enforcement gap for non-staff customer admins.

## What Was Done

Applied a two-step transform (import swap + callsite swap) to every file classified `staff-only` in `01.1-CLASSIFICATION.md`. The pattern is identical in all 23 files:

```
import { requireAdmin } from '@/lib/api-auth'
→
import { requireStaff } from '@/lib/api-auth'

const { error } = await requireAdmin()
→
const { error } = await requireStaff()
```

Two files used `{ error, session }` destructuring (session used in handler body); that was preserved as-is since `requireStaff` returns the same shape.
