    - Implement getEntryTypeSummaryForProject(projectKey: string, releaseIds: string[]):
      * If releaseIds.length === 0 → return empty Map<string, EntryTypeCounts>()
      * Single Drizzle query: select releaseId + linkType from releaseLogLinks where releaseId IN (releaseIds)
      * Walk results, build per-release counts: linkType='bug' → fixes++; linkType='feature' → features++; linkType='external' → no increment (external links are NOT typed counts; the consumer treats releases with no fix/feature links as "other" bucket on the client)

      * Get those releaseIds, call getEntryTypeSummaryForProject(projectKey, devReleaseIds) to bucket
      * Aggregate: totalEntries = count of dev rows since prod (count of releases, NOT entries[] within them — Phase 14 customer card counts RELEASES with type-attribution, per CONTEXT.md decision "counts derived from same release_log_links join used by filter chips")

      * Return { totalEntries, fixes, features, other, hasDelta: true, oneliner }
