Export `GET(req, { params }: { params: Promise of { slug } })`:

1. requireSignedIn(); on error return error.
2. Await params (slug).
3. getCurrentUserContext(session); 403 if null.
4. Look up project by `eq(projects.key, slug)`. 404 if not found.
5. Membership 404-no-leak: `isMember = ctx.isStaff || ctx.memberships.some(m => m.project_key === project.key)`. If not, 404.
6. Parse pagination params from `new URL(req.url).searchParams`:

7. Fetch using Drizzle relational query (same shape as page.tsx):
   ```
   const rows = await db.query.releaseLogs.findMany({
     where: eq(releaseLogs.project, project.key),
     with: {
       feedback: { orderBy: (f, { asc }) => [asc(f.createdAt)] },
       approvals: { orderBy: (a, { desc: desc2 }) => [desc2(a.approvedAt)] },
     },
     orderBy: [desc(sql`coalesce(${releaseLogs.deployedAt}, ${releaseLogs.releasedAt})`)],
     limit: limit + 1,
     offset,
   });
   ```
