### Task 2: Dashboard membership filter

`src/app/admin/page.tsx` modified:

- **Import additions:** `getCurrentUserContext` from `@/lib/auth-context`; `inArray` added to drizzle-orm imports
- **`getDashboardStats(projectKeys: string[] | null)` signature change:**
  - `projectKeys === null` ‚Üí staff / DB-error fallback; all 7 `db.select()` calls run unfiltered (`where(undefined)`)
  - `projectKeys === []` ‚Üí non-staff with no memberships; returns immediate `{0, 0, 0, 0, []}` short-circuit
  - `projectKeys.length > 0` ‚Üí non-staff scoped; 4 per-table filter constants (`inArray(table.project, projectKeys)`) combined with existing `openBugsCondition` / `pendingFeaturesCondition` via `and()`
- **`AdminDashboard` prelude refactor:** replaced `Promise.all([session, getDashboardStats()])` with sequential session ‚Üí `getCurrentUserContext` ‚Üí compute `projectKeys` ‚Üí `getDashboardStats(projectKeys)`
- **JSX layout unchanged:** modules array, stat cards, Project Health grid are byte-for-byte identical to pre-task state
